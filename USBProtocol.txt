Commands Syntax:
"Command;Type;Value"

Commands:
- "version":	Show Application Version 	(Value not used);
- "led":	Control Led Task and Blink Time	(Value = Blink Time);
- "go":		Start IMU Sample		(Value = Total Time).

Type:
- "0": Get Value;
- "1": Set Value;
Parameter only matter for values configuration.

Value:
- "##.###": Float Value
Parameter only matter for values configuration.
